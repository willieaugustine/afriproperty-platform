# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

**Please DO NOT open a public issue for security vulnerabilities.**

Instead, email us at: security@afriproperty.io

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will respond within 48 hours and keep you updated on the fix progress.

## Bug Bounty

We offer rewards for responsibly disclosed security issues:
- Critical: $5,000 - $50,000
- High: $2,000 - $10,000
- Medium: $500 - $2,000
- Low: $100 - $500

More details: https://afriproperty.io/bug-bounty
